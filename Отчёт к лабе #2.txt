1.	Постановка задачи
Задано семейство множеств букв. Найти такое k, для которого можно построить
множество, состоящее из k букв, причем каждая из них принадлежит ровно k
множествам заданного семейства.
2.	Разработка программы
2.1.	Модель и метод решения задачи
Во-первых, обратим внимание на то, что в каждом множестве по определению не может быть повторяющихся элементов.
Последовательно проверив все множества, посчитаем количество вхождений каждой буквы и запишем полученные значения в массив.
Затем, найдём в массиве такое число n, которое встречается в нём ровно n раз. n – это и есть искомое k.
2.2.	Описание структур данных
В программе использовался массив для хранения количества вхождений букв в множества. Кроме того, использовались такие структуры данных как множество (std::set), вектор (std::vector), объекты класса std::string и текстовый файл – все они использовались лишь для реализации условия поставленной задачи или удобства представления, но не имеют влияния на метод решения.
2.3.	Алгоритм решения задачи
1.	Считываем из файла все множества в виде строк, которые записываем в вектор.
2.	Создаём массив из 26 целочисленных элементов (по количеству букв в английском алфавите) и заполняем его нулями.
3.	Проходя по вектору, увеличиваем на единицу элемент массива с номером a-96, где a – код встреченного символа в векторе, 96 – это код символа ‘a’. Таким образом каждому символу алфавита соответствует номер от 0 до 26 в массиве.
4.	Упорядочиваем массив методом пузырька.
5.	Перебираем числа в массиве, увеличивая счётчик currentEntryAmount пока встречаются одинаковые не нулевые числа.
6.	Когда встречено число a[i] ≠ a[i-1], если currentEntryAmount равно a[i-1], останавливаем поиск.  k = a[i-1].
7.	Иначе, обнуляем currentEntryAmount и продолжаем перебор.
8.	Выводим на экран найденное k.

3.	Руководства пользователя
Введите количество множеств.
4.	Руководство программиста
Программа использует в качестве входного текстовый файл, формирующийся случайным образом.
Сложность выполнения решения задачи линейна относительно количества множеств.
В случае ошибок работы с файлом программа выводит сообщение на экран.
